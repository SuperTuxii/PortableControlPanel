# PortableControlPanel
A control panel with a touchscreen, speaker and more that magnetically connects to a stand. The builtin battery allows it to be taken around and used wirelessly. Placing it on the stand will charge the battery through the magnetic connectors. It can be used to control lights or as a macro pad etc.

I made this since my light switch is quite far from my bed and it's always annoying when I'm in bed and have to turn off the lights. So I will probably put this in my bed for that purpose, but making it wireless still leaves me with the possibility to use it in even more ways in the future.
